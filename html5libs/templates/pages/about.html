About {{ SITE_TITLE }}

{{ SITE_TITLE }} addresses the problem in the community of being able to easily identify and compare good client-side software.

What is a modern client-side library?

A modern client-side library is anything that solves a problem for you and/or others in the open web community that can be stored on a repository such as Github or Bitbucket. {% comment %} If it can be stored in PyPI, that's even better! {% endcomment %}

When did you code this?

This project is made possible by Open Comparison, which was conceived for and coding during Django Dash 2010. Since then we've been feverishly adding more features, improving the design, and trying to make data entry as easy/intuitive as possible.

Who built this thing?

Daniel Greenfeld and Audrey Roy are the creators of this project. Additional contributing members are detailed in the team page. This version is a fork that doesn't currently aim to maintain compatability.

Can I help?

Open Comparison is a volunteer based open source effort. Please submit any ideas or issues and also fork and submit back patches and enhancements.
